SupplyLine MRO Suite - Installation Package 
============================================= 
 
This package contains the SupplyLine MRO Suite application. 
 
INSTALLATION OPTIONS: 
 
Option 1 - Portable Use: 
  - Navigate to the SupplyLine-MRO-Suite folder 
  - Double-click "SupplyLine MRO Suite.exe" to run 
  - No installation required! 
 
Option 2 - Full Installation: 
  - Right-click "install.bat" and select "Run as administrator" 
  - Follow the prompts to install to Program Files 
  - A desktop shortcut will be created 
 
UNINSTALLATION: 
  - Run "uninstall.bat" as administrator to remove the application 
 
SYSTEM REQUIREMENTS: 
  - Windows 10 or later 
  - 4GB RAM minimum 
  - 500MB free disk space 
 
For support, please contact the SupplyLine MRO Suite team.
